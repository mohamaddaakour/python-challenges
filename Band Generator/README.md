# Band Name Generator

## Description
The **Band Name Generator** is a simple Python command-line program designed to create a unique and fun band name for the user. It utilizes string manipulation to combine specific user inputs—the city they grew up in and the name of their pet—to generate a personalized result.

This project is often used as a beginner exercise to understand Python input/output (I/O) and string concatenation.

## Features
* **Interactive CLI:** Prompts the user for input via the console.
* **String Concatenation:** Combines multiple strings into a single variable.
* **User-Friendly Output:** Displays a clear welcome message and the final generated name.

## Prerequisites
* Python 3.x installed on your machine.
